# Python Quiz Application

## Description
A simple command-line quiz application built using Python. The program allows users to take quizzes from multiple categories such as Geography, Maths, and Science. Questions appear in random order, inputs are validated, scores are calculated with positive and negative marking, and total time taken is recorded.

## Features
- Multiple quiz categories
- Randomized question order
- Input validation
- Score calculation
- Time tracking
- Final consolidated score summary

## Quiz Categories
- Geography Quiz
- Maths Quiz
- Science Quiz

## Scoring System
| Result | Points |
|-------|--------|
| Correct Answer | +10 |
| Wrong Answer | -5 |
