A simple yet powerful **command-line quiz application** built using Python.  
Test your knowledge across multiple categories while tracking your **score 🎯**, **accuracy 📊**, and **time ⏱️**.

## 🚀 Features

- 🧠 **Multiple Quiz Categories**
  - 🌍 Geography
  - ➗ Maths
  - 🔬 Science

- 🔀 **Randomized Questions**
  - Questions appear in random order every time

- ✅ **Input Validation**
  - Prevents invalid inputs
  - Ensures smooth user experience

- 🎯 **Smart Scoring System**
  - Positive and negative marking

- ⏱️ **Time Tracking**
  - Measures total quiz duration

- 📊 **Final Score Summary**
  - Displays complete performance at the end

## 📌 How It Works

| Step | Description |
|------|------------|
| 1️⃣ | User selects quiz category |
| 2️⃣ | Questions are shuffled randomly |
| 3️⃣ | User answers each question |
| 4️⃣ | Input is validated |
| 5️⃣ | Score is updated dynamically |
| 6️⃣ | Final results displayed |

---

## 📊 Scoring System

| Result | Points |
|--------|--------|
| ✅ Correct Answer | +10 |
| ❌ Wrong Answer | -5 |
